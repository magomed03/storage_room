Установить компилятор harbour (clipper) для тех у кого Windows

1. Переходим по ссылке
https://sourceforge.net/projects/harbour-project/files/

2. Нажимаем на Download Latest Version

3. Скачанный файл запускаем, harbour устаналивается

4. Проверяем, что harbour установлен в c:\hb30 (что папка есть)

5. Открываем коммандную строку. Чтобы из любой папки был виден hbmk2, нужно один раз выполнить данную команду:
setx /M path "%path%;c:\hb30\bin"

6. Закрываем коммандную строку

7. В файловом менеджере (проводник) создаем папку c:\clipper (либо с иным наименованием), в которой будем создавать и хранить все программы и файлы базы данных

Процесс подготовки завершен

-----------------------------------------------------------

Для тех у кого apple osx установка отличается

1. Нужно установить brew (если не установлен) - менеджер приложения для mac:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

2. После установки brew, через него нужно установить harbour:
brew install harbour

3. После установки harbour компилятор будет доступен из любой папки 

Процесс подготовки завершен

------------

Программы пишем в любом редакторе (например, в Dev-C++), сохраняем в папке c:\clipper (либо в любой в которой удобно)

В коммандной строке переходим в папку c:\clipper (либо в ту, в которой создаем програмы):
cd c:\clipper


Компилировать программу ИМЯ.prg:
в командной строке hbmk2 ИМЯ.prg

Запустить программу:
в командной строке ИМЯ
     (в зависимости от конкретного компьютера возможны варианты: иногда срабатывало ИМЯ, ./ИМЯ)

Чтобы посмотреть, что имеется в папке:
dir
